Twilio = the phone infrastructure
Retell AI = the brain of the conversation
Here’s what happens step by step:
1️⃣ A customer calls your Twilio number
2️⃣ Twilio routes that call to Retell AI
3️⃣ Retell AI handles the conversation (ask questions, qualify, respond)
4️⃣ The AI can then trigger actions — book appointments, update CRM, send follow-ups
Twilio handles the telecom layer.
Retell handles intelligence.
When you integrate this with automation tools (like n8n), it becomes even more powerful:

– Internal team notified →
– Follow-up sequence triggered

### 4. 核心代码流程

```python
def handle_turn(self, state: SessionState, text: str) -> Dict:
    # Step 1: Router 分类意图
    self.engine.run_node("router", state, text)
    
    # Step 2: Supervisor 决策
    self.engine.run_node("supervisor", state, text)
    decision = self.agents["supervisor"].decide(state)
    
    # Step 3: 根据决策执行
    if decision == "clarify":
        response = "I want to make sure I understood..."
    elif decision == "handoff":
        self.handoff.transfer(state.handoff_reason)
        response = "Let me connect you to a person."
    else:
        # Step 4: Task Agent 执行任务
        _, response = self.engine.run_node("task", state, text)
    
    return {"state": state, "response": response}
```

### 5. Agent 职责总结

| Agent | 职责 |
|-------|------|
| **Router** | 意图分类、患者类型识别、置信度评估 |
| **Supervisor** | 质量监控、错误处理、转人工决策 |
| **Task** | 核心业务逻辑：预约、注册、报告生成 |
| **Context** | 上下文管理、对话历史维护 |
| **Audit** | 合规审计、日志记录 |
