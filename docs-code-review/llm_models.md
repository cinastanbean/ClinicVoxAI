### 模型演进的核心趋势

1. **速度演进**：从GPT-3.5 Turbo的亚秒级响应（0.4-0.5秒），到GPT-4o的232毫秒音频响应，再到GPT-5.3-Codex-Spark的超1000 tokens/秒，实时性持续突破。

2. **应用专业化**：Codex系列专注编程、Whisper专注语音、DALL-E/Sora专注视觉生成、o系列专注深度推理。

3. **成本优化**：通过mini/nano版本持续降低API调用成本，GPT-4.1系列比GPT-4o便宜80%。

4. **开源探索**：GPT-OSS系列（2025年）是OpenAI自GPT-2以来首次开源权重模型。
